AI-Powered Brain MRI Tumor Detection & Classification

Project Overview

NeuraSight is a medical imaging platform that classifies brain tumors from MRI scans using deep learning. Built with a microservices architecture — React frontend, Express.js API gateway, and FastAPI ML service — it provides real-time classification across 4 tumor types with Grad-CAM explainability heatmaps.

The model uses EfficientNet-B0 from the timm library, pre-trained on ImageNet and fine-tuned on 5,600 brain MRI images. Trained on Google Colab (Tesla T4 GPU) for 10 epochs using AdamW optimizer, achieving 95% test accuracy across 1,600 test images.

Model Architecture

timm EfficientNet-B0 (Transfer Learning)

The system uses timm.create_model('efficientnet_b0', pretrained=True, num_classes=4) — a PyTorch Image Models implementation of EfficientNet-B0 pre-trained on ImageNet (1.2M images), fine-tuned on 5,600 brain MRI images for 4-class tumor classification.

Architecture: Stem Conv → 16 MBConv Blocks (with Squeeze-and-Excitation) → Head Conv (320→1280) → Global Avg Pooling → Linear(1280, 4)

Training Config: AdamW optimizer (lr=0.0001), CrossEntropyLoss, 10 epochs, batch size 32, ImageNet normalization (mean=[0.485,0.456,0.406], std=[0.229,0.224,0.225])

Key Features:

  • Transfer Learning from ImageNet (1.2M images)
  • Only 5.3M Parameters (lightweight for deployment)
  • Compound Scaling (width × depth × resolution)
  • Squeeze-and-Excitation Attention
  • Single Linear Classifier Head (1280→4)
  • Data Augmentation (flip, rotation, color jitter)

Explainable AI

Grad-CAM Visualization

Gradient-weighted Class Activation Mapping (Grad-CAM) targets the last convolutional block (model.blocks[-1]) of the timm EfficientNet-B0. Gradients are backpropagated for the predicted class, globally average-pooled to get channel importance weights, then used to weight the feature maps. After ReLU and normalization, the heatmap is overlaid on the original MRI with JET colormap at 40% opacity.

Benefits:

  • Highlights Tumor Region the Model Focused On
  • No Architecture Modification Required
  • Builds Clinician Trust in AI Decisions
  • Generated in <0.5 Seconds Per Image

Future Expansion

The NeuraSight platform is designed with extensibility in mind. Future development plans include expanding the system to support additional medical imaging modalities and diagnostic applications:

  • CT Scan Analysis: Extend deep learning models to analyze computed tomography scans for various pathologies including lung nodules and abdominal abnormalities.
  • X-Ray Diagnostics: Implement chest X-ray analysis for pneumonia detection, COVID-19 screening, and other pulmonary conditions.
  • Retinal Imaging: Develop models for diabetic retinopathy detection and age-related macular degeneration screening from fundus photographs.
  • Skin Lesion Detection: Create dermatological AI for melanoma and skin cancer detection from dermoscopic images.
  • Multi-Modal Fusion: Integrate multiple imaging modalities for comprehensive diagnostic insights and improved accuracy.
  • Real-Time Inference: Optimize models for edge deployment and real-time clinical decision support in hospital environments.

Ethical Considerations

This system is designed as a clinical decision support tool, not a replacement for professional medical judgment. All predictions should be validated by qualified healthcare professionals. The explainable AI component ensures transparency in the decision-making process, supporting informed clinical decisions.
